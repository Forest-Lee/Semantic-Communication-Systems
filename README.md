# Deep-Learning-Enabled-Semantic-Communication-Systems-with-Task-Unaware-Transmitter-and-Dynamic-Data
pytorch implementation of "Deep Learning-Enabled Semantic Communication Systems with Task-Unaware Transmitter and Dynamic Data"

## Prerequites
* [Python 3.7]
* [PyTorch 0.1.12]
* [Torchvision 0.9.1]
* [Torch 1.5.1]
* [Numpy 1.21.2]

## The folders
["semantic_extraction"](semantic_extraction) and ["Semantic System with DA"](Semantic System with DA) are the proposed method of the semantic extraction and the data adaptation parts in the paper.
